# Cafeteria do Yan - Protótipo Web
Este repositório contém o protótipo da página web "Cafeteria do Yan", desenvolvido em HTML5, CSS3 e Bootstrap. O objetivo é apresentar uma interface simples, moderna e funcional, que simula a página institucional de uma cafeteria.
# Descrição do Projeto
O projeto consiste em uma página estática que exibe informações sobre a cafeteria, tipos de cafés oferecidos, imagem ilustrativa e botões de ação. O foco está na experiência visual e na organização do conteúdo, utilizando estilos personalizados com paleta de cores inspirada em tons de café.
# Tecnologias Utilizadas
HTML5
CSS3
Bootstrap 5.3.2 (CDN)
Google Fonts - Poppins
Imagens locais e externas

# Dependências
Este projeto utiliza apenas dependências externas via CDN:
Bootstrap: https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css
Não há bibliotecas adicionais nem pacotes para instalar.
# Como Executar o Projeto
Baixe ou clone o repositório: git clone https://github.com/seu-usuario/cafeteria-do-yan
abra o arquivo index.html diretamente no navegador.
# Funcionalidades do Protótipo
Exibição de descrição da cafeteria
Lista de tipos de café oferecidos
Imagem central com sombra e bordas arredondadas
Botão de ação "Fazer Pedido"
Botão de contato
Layout responsivo
Background fixo com efeito blur
# Responsividade
O site se adapta automaticamente para telas menores, utilizando recursos nativos do Bootstrap e media queries no CSS.
# Observações Importantes
Certifique-se de que a imagem usada como fundo possui boa resolução.
O sistema atual não inclui funcionalidades dinâmicas (JavaScript).
Os botões são demonstrativos e não redirecionam para outra página.
